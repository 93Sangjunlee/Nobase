익스텐션 소개
1. Auto Rename Tag : HTML 태그 수정 시 시작 태그와 종료 태그를 자동으로 함께 수정해 주는 모듈
2. Code Spell Checker : 사전에 없거나 자주 사용되지 않는 단어가 사용되면 경고해 주는 모듈
3. Error lens : 문제가 있는 부분을 직관적으로 알려주는 모듈
4. Live Server : html을 작성 할 때 변경된 점을 바로 확인할 수 있도록 도와주는 모듈

파일트리
부모자식 : 상위 폴더가 하위 폴더를 담고있는 형태
형제 : 폴더나 파일은 같은 depth에 위치한 형태 - 하나의 부모 폴더 안에 있는 동등한 파일들
