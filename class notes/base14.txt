아이프래임 / 프레임
복잡하니 주의

아이프레임은 웹 페이지 안에 다른 웹페이지를 삽입하는 기법이다.
<ifream src="http://링크" frameborder="0" allowfullscreen></ifream>
위의 예제와 같이 입력하게 되면 된다.
ifream은 태그명
src : 불러올 페이지의 URL
scrolling : 아이프레임 안에서 스크롤링을 허용할지의 여부이다. auto(자동)), yes(항상 노출), no(항상 노출하지 않음)
ex14를 참고하자.

프레임 내에서 폭, 너비 등의 일반적인 속성들은 css를 활용해서 컨트롤 하는것이 권장된다.

예를 들어 페이스북의 좋아요 등이 페이지 내에 삽입 된다면
페이스북 내에서 페이스북이 제공하는 웹페이지를 통해 해당 링크의 글을 좋아요 누른 것과 동일한
효과를 얻는다.
페이스북 등 좋아요 누르는 방법 알면 좋을 것 같다.

프레임은 웹 페이지의 조각난 구멍에 웹 페이지를 삽입하는 것
프레임은 삽입된 페이지들이 빈틈없이 연결되어 있어야 한다.
프레임은 html5로 진화하면서 권장되지 않는 태그들로 구분되었다.
